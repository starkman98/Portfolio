# Anton Stark — Portfolio

A personal portfolio website built with React, TypeScript, and Tailwind CSS. Showcases my background, skills, projects, and homelab setup.

## Tech Stack

- **React 19** with TypeScript
- **Vite** — build tool & dev server
- **Tailwind CSS v4** — utility-first styling
- **Framer Motion** — animations
- **react-icons** — icon library

## Sections

- **Hero** — Introduction and contact links
- **About** — Background and background story
- **Skills** — Technical and soft skills
- **Projects** — Highlighted personal and school projects
- **Homelab** — Self-hosted infrastructure overview
- **Contact** — Contact form and links
